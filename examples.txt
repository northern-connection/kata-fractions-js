Examples
========

Mismo denominador:
1/1 + 0/1 = 1/1
1/2 + 2/2 = 3/2

Denominadores distintos
1/2 + 1/1 = 3/2
1/2 + 1/3 = 3/6 + 2/6 = 5/6

Numerador mayor que el denominador
3/2 + 5/2 = 8/2 = 4/1
3/2 + 4/3 = 9/6 + 8/6 = 17/6

Simplificable
3/2 + 5/2 = 8/2 = 4/1
1/2 + 1/2  = 2/2 = 1/1

Resta
1/2 - 1/2 = 0/2
-1/2 + 3/2 = 1/2
-1/2 - 3/2 = -4/2 = -2/1

Casos de error
1/0 + 1/0 = infinito
